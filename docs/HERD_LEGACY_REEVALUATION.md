# 기존 HERD 모델 재평가

상태: `REJECTED_AS_NEXT_GENERATION_BASELINE`  
평가 버전: `2026.07-v1`  
평가일: 2026-07-19

기존 HERD v4와 Python 연구용 v6.1 행동 규칙을 동일 자본, 동일 기간,
다음 거래일 시가 체결, 동일 비용 조건으로 다시 평가했다. 최종 목표는
비용 차감 후 장기 Buy & Hold를 이기는 것이다.

## 평가 범위

- 현재 고정 유니버스 55종목
- 종목별 최근 10년 원천 가격 요청
- 모든 v4 장기 지표가 준비된 시점부터 평가
- 초기 자본 100%, 다음 거래일 시가 체결
- 기본 비용: 수수료 0.10% + 슬리피지 0.05%
- 스트레스 비용: 수수료 0.20% + 슬리피지 0.15%
- v4: 가격 기반 기본 점수만 사용
- v6.1: Python 연구 규칙을 재구성해 사용

PIT EPS와 당시 섹터 보정은 신뢰할 수 있는 과거 데이터가 없어 제외했다.
Java 운영 v6.1과 Python 연구 v6.1의 계산식이 완전히 같지 않으므로 이번
결과를 운영 코드의 정확한 재현으로 해석하면 안 된다.

커졌다. 현재 규칙을 차세대 HERD로 승격하지 않는다.

## 판정

- v4: `FAIL`
- Python v6.1: `FAIL`
- 운영 HERD 교체: `금지`
- 차세대 후보의 비교 기준: v4, v6.1, Buy & Hold를 모두 유지

이번 실패는 HERD 개발 중단을 의미하지 않는다. 다음 단계에서는 기존
점수의 가중치를 미세 조정하지 않고 참여, 추세·상대강도, 위험, 기업 상태를
각각 독립 검증한다. 독립 효과가 확인되지 않은 증거군은 차세대 후보에
포함하지 않는다.

## 남은 한계

  못했다.
