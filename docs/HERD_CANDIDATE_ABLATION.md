# 차세대 HERD 후보 Ablation

상태: `ALL_CANDIDATES_REJECTED`  
평가 버전: `2026.07-v1`

6단계에서 독립 효과가 확인되지 않은 증거군을 단순 합산하지 않고 역할을
분리해 B0~B4 후보를 구성했다.

## 후보 구조

| 후보 | 역할 |
|---|---|
| B0 | 가격 기반 v4를 목표 노출 방식으로 표현한 비교 기준 |
| B1 | 중복 제거된 추세·상대강도 코어 |
| B2 | B1 + 참여 확인 조건 + Healthy/Exhausted Rush |
| B3 | B2 + 독립 위험 노출 상한 |
| B4 | PIT 기업 상태 데이터 부재로 차단 |

참여는 약한 추세의 감축 확인에만 사용했고, 위험은 HERD 방향 점수에 더하지
않고 최대 노출 상한으로만 사용했다. Healthy Rush는 강한 추세와 참여가
함께 유지될 때 전액 노출을 유지하며, 참여가 약한 Rush만 85%로 낮췄다.
