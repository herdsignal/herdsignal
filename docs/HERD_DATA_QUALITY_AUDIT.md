# HERD 데이터 품질 감사

상태: `PARTIAL_PRICE_ONLY`  
감사일: 2026-07-19  
유니버스: `2026.07`, 55종목  
기간: 2016-07-18~2026-07-17

## 결론

현재 데이터는 **가격 기반 연구는 가능하지만, 생존자 편향을 제거한 최종
모델 검증과 시점 정합 재무 모델에는 사용할 수 없다.**

| 영역 | 상태 | 판정 |
|---|---|---|
| 55종목 조정 OHLCV | 55/55 통과 | 가격 연구 가능 |
| 분할·배당 | 조정 가격에 포함 | 총수익 근사 가능, 사건별 감사 불가 |
| 원시 종가·배당 현금흐름 | 저장 안 함 | 정밀 성과 귀속 불가 |
| 과거 지수 구성 | 레코드 0개 | 생존자 편향 미해결 |
| 상장폐지·합병·파산 표본 | 0개 | 최종 일반화 검증 불가 |
| 과거 EPS 발표일·컨센서스 | 신뢰 데이터 없음 | 기업 상태 증거군 사용 불가 |

## 가격 검사 결과

55종목 모두 다음 조건을 통과했다.

- 10년 2,514행
- 날짜 중복·역순 없음
- OHLCV 결측 없음
- 음수·0 이하 가격 없음
- OHLC 범위 오류 없음
- 음수 거래량 없음
- 최근 가격 기준일 2026-07-17
- 감사일 대비 2일 지연
- 최대 달력 공백 4일

배당 조정 과정의 `1e-14` 수준 부동소수점 차이를 실제 OHLC 오류로 오인하지
않도록 가격 대비 `1e-8` 허용오차를 적용했다. 의미 있는 가격 역전은 계속
실패 처리한다.

## 기업행동

현재 수집기는 yfinance `auto_adjust=True`를 사용한다.

- 주식분할과 배당 영향이 조정 OHLC에 반영된다.
- Buy & Hold와 HERD가 같은 조정 가격을 사용하므로 상대 비교는 가능하다.
- 원시 종가, 배당액, 분할 이벤트가 저장되지 않아 결과를 가격수익과
  배당수익으로 나눠 검증할 수 없다.

따라서 2단계 비교 엔진의 가격 기반 재평가는 진행할 수 있지만
`buy_hold_total_return_attribution_ready`는 `false`로 유지한다.

## 재무 데이터

Finnhub 무료 EPS 응답의 `period`는 분기 종료일이며 실제 발표일이 아니다.
현재 저장소에는 다음을 함께 복원한 데이터셋이 없다.

- 실제 발표일
- 당시 발표된 실제 EPS
- 발표 직전 컨센서스 EPS
- 수정 전 재무 수치
- 데이터 출처

이 값이 준비되기 전까지 B4 기업 상태 후보와 역사 EPS 승수는 제외한다.

## 생존자 편향

`data/herd/universe_history.csv`에는 헤더만 있고 레코드가 없다. 현재 55종목은
현존 대형주 중심의 고정 표본이다.

필요한 데이터:

- 날짜별 S&P 500 구성 종목
- 당시 섹터
- 편입·편출일
- 합병·상장폐지·파산 사유
- 해당 종목의 편출 이후 가격 또는 최종 청산 처리
- 검증 가능한 출처

이 자료가 없으므로 `survivorship_safe_validation_ready`는 `false`이며,
최종 모델 승격 게이트는 통과할 수 없다.

## 다음 단계에 미치는 영향
